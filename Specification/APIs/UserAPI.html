发送用户注册短信

POST

URL

/GlobalAccount/SendSMS

Parameters

 
AخA
{
    "PhoneNumber": <string 手机号>,
}

Results

 
xxxxxxxxxx
{
    "Succeeded": <bool 是否成功(下同)>,
    "ErrorMessage": <string 错误信息(下同)>
}

该接口只能每隔50秒调用一次, 小于50秒会发送错误信息, 建议前端显示时每隔60秒才能重新发送短信

验证用户登录并获取用户所有信息

POST

URL

/GlobalAccount/IsAuthenticated

Parameters

Results

 
xxxxxxxxxx
{
    "Succeeded": <bool>,
    "Data": {
        "Id": <string 用户编号>,
        "Email": <string>,
        "PhoneNumber": <string 手机号>,
        "UserName": <string 用户名>,
        "CustomerInfos": [{
            "Hotel": {
                "Id": <int 饭店编号>,
                "Name": <string 饭店名>
            },
            "Points": <string 该饭店的积分>,
            "VipLevel": {
                "Id": <int 会员等级编号>,
                "Name": <string 会员等级名>
            },
            "DinesCountId": <int 该饭店点过的点单数量>,
        }, ...]
    }
}

如Succeeded为false, Data为null 如会员没有去过某饭店, 则CustomerInfos中没有该饭店信息 如会员不是该饭店会员, 则CustomerInfos中该饭店的VipLevel为null
